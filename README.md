# ECG-Arrhythmia-Classification-using-LSTM-and-BiLSTM
θα χρησιμοποιηθεί το MIT-BIH Arrhythmia Database, και θα γίνει σύγκριση των μοντέλων: LSTM και Bidirectional LSTM.
Θα γίνει σύγκρισή τους ως προς το πόσο καλά ταξινομούν τα είδη αρρυθμιών του dataset, ενώ θα δοκιμαστούν διαφορετικές τιμές για διάφορες παραμέτρους, όπως learning rate,hidden size.

# MIT-BIH Arrhythmia Dataset
Το MIT-BIH Arrhythmia Dataset της βάσης PhysioNet περιλαμβάνει 48 καταγραφές ECG διάρκειας 30 λεπτών με συχνότητα δειγματοληψίας 360Hz από συνολικά 47 ασθενείς, καλύπτοντας τόσο φυσιολογικούς όσο και παθολογικούς καρδιακούς ρυθμούς. Οι 23 καταγραφές επιλέχθηκαν ώστε να είναι αντιπροσωπευτικές καθημερινών κλινικών συνθηκών, ενώ άλλες 25 επιλέχθηκαν στοχευμένα για να περιλαμβάνουν σπάνιες αλλά κλινικά σημαντικές αρρυθμίες και διαταραχές.

Στην πλειονότητα των εγγραφών χρησιμοποιούνται οι απαγωγές MLII και V1, όπου η MLII διευκολύνει τον εντοπισμό φυσιολογικών συμπλεγμάτων και η V1 την παρατήρηση παθολογικών παλμών. Στο dataset υπάρχουν συχνά πάνω από 2 κανάλια για κάθε ασθενή, ωστόσo, συνήθως χρησιμοποιείται μόνο το πρώτο κανάλι (MLII). Για τον λόγο αυτό εξαιρέθηκαν  οι ασθενείς 102, 104 και 114, οι οποίοι δεν έχουν το MLII πρώτο κανάλι.
